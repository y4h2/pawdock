# Deskpet

桌面宠物应用，使用 Go + Ebitengine 开发，透明无边框窗口，角色通过 sprite sheet 动画呈现。

## 技术栈

- 语言：Go
- 游戏引擎：Ebitengine (ebiten/v2)
- 资源嵌入：`//go:embed material/*`
- 配置：deskpet.ini（INI 格式）
- 音频：ebiten/audio + WAV
- 字体：bitmapfont

## 项目结构

- `main.go` — 全部代码（单文件，尚未拆分模块）
- `material/` — 嵌入资源（sprite sheet PNG、图标 PNG、音效 WAV）
- `docs/` — 设计文档（功能路线图、工作模式设计、spritesheet 流程、存储设计）
- `deskpet.ini` — 运行时配置

## 当前游戏玩法

### 角色状态（State）

| 状态 | 说明 | 动画 | 行为 |
|------|------|------|------|
| Idle | 默认待机 | idle（121帧，循环） | 面朝鼠标方向，following 模式下靠近鼠标时停下 |
| Walk | 行走跟随 | walk（73帧，循环） | 8方向移动追踪鼠标，距离<200px 时切回 Idle |
| Happy | 开心 | happy（73帧，单次） | 菜单触发或工作模式休息时播放，结束回 Idle |
| Pick | 拖拽 | pick（73帧，循环） | 鼠标按住拖动超过5px触发，松开回 Idle |
| Play | 玩耍 | play（121帧，单次） | 菜单触发或长休息时播放，结束回 Idle |
| Work | 工作专注 | work（145帧，循环） | 工作模式专注阶段，面朝屏幕中心，不跟随鼠标 |
| Hide | 躲藏 | hide（73帧，单次） | 躲猫猫开始阶段，播完后移到屏幕边缘进入 Peek |
| Peek | 探头 | peek（73帧，循环） | 从屏幕左/右边缘探出，鼠标靠近会换边，3次后停下等点击 |
| Found | 被发现 | found（73帧，单次） | 用户点击找到后播放，结束回到原位 Idle |

### 交互方式

1. **鼠标跟随**：默认开启，角色会走向鼠标位置，靠近后停下
2. **拖拽移动**：按住鼠标拖动窗口，角色播放 pick 动画
3. **右键菜单**：点击角色弹出圆形图标菜单（5个按钮纵向排列在角色右侧）
   - Happy：触发开心动画
   - Play：触发玩耍动画
   - Peekaboo：躲猫猫小游戏
   - Follow/Stay：切换跟随/原地待命模式
   - Work：启动/停止工作模式（番茄钟）

### 躲猫猫（Peekaboo）

1. 点击菜单 Peekaboo 按钮，角色播放 Hide 动画
2. 角色移到屏幕左/右边缘，只露出部分身体（Peek 状态）
3. 鼠标靠近时角色换到另一边，共躲3次
4. 第3次后不再躲避，用户点击即可"找到"
5. 找到后播放 Found 动画，回到原来的位置

### 工作模式（番茄钟）

- 经典番茄钟循环：专注 25min -> 短休息 5min -> 循环4轮 -> 长休息 15min
- **专注阶段**：角色播放 Work 动画，停止跟随鼠标，面朝屏幕中心
- **短休息**：角色播放 Happy 动画，恢复鼠标跟随，气泡提醒喝水/休息
- **长休息**：角色播放 Play 动画，气泡庆祝完成一轮
- **倒计时显示**：角色右上方显示 `分:秒` 数字（专注时红色，休息时绿色）
- **气泡对话**：状态切换时角色头顶弹出随机文字气泡（4-5秒后淡出）
- **音效提醒**：专注结束时播放提示音（notify.wav）
- 工作模式中可随时通过菜单停止

### 配置项（deskpet.ini）

```ini
Speed = 1              # 移动速度
Scale = 0.5            # 窗口缩放比例
FocusDuration = 25     # 专注时长（分钟）
ShortBreak = 5         # 短休息（分钟）
LongBreak = 15         # 长休息（分钟）
CyclesBeforeLong = 4   # 几轮后长休息
```

### Sprite Sheet 规范

- 每帧 256x256，9列排列
- 加载函数：`loadSheetFrames(path, 256, 256, 9, totalFrames)`
- 动画播放：每3 tick 换一帧（TPS=50，即约 16.7fps）
- 素材流程：视频(480x640,24fps,绿幕) -> 12fps抽帧 -> 去背景 -> 256x256归一化 -> 9列 sprite sheet
